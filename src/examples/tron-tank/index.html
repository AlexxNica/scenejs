A COLLADA Tron Tank Model

This scene contains the Tank Program from the 1982 Disney motion picture "Tron".

Mouse wheel controls speed, left/right drag to steer, up/down drag to change viewpoint height.

The tank model is a SceneJS content module which was pre-translated from COLLADA.

Tank model is courtesy of Abraham Katase, originally sourced from the Google 3D Warehouse here
