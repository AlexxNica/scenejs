Data Flow - Adding Elements within a Subgraph

In this example we're using a SceneJS.WithConfigs node to add a teapot to its subgraph.

This isn't terribly dramatic - you should just see a teapot appear!

This is some powerful stuff - the SceneJS.Socket node contains the same functionality as a SceneJS.WithConfigs, providing the means for a server to push content into a client scene graph.
