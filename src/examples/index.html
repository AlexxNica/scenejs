SceneJS Examples

SceneJS is a JavaScript framework that allows you to functionally program interactive 3D scenes for browsers that support the WebGL canvas element. These example scenes show off core features of SceneJS.

Basic

Basic no-frills static scene graphs to give you the basic SceneJS idea.

Animation

Scenes with movement.

Generation

Scenes that are procedurally generated with generator nodes.

Remote Asset Loading

Scenes that employ asset nodes to load parts of themselves from remote locations.
