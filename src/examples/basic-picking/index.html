SceneJS Picking - Firing and Catching "picked" Events

This example shows how "picked" events are created within a scene graph when you click on its various geometries.

You can define "picked" listeners on any kind of node so that they will catch those events when they occur on Geometry nodes within their subgraphs.

On mouse down, we render the scene in picking mode, passing in the mouse canvas coordinates. This will cause a scene render traversal in which all the "picked" listeners will fire on those nodes situated above whatever geometry node was picked. Each of those node's listeners will be called as their node is visited, in the usual depth-first, right-to-left traversal order.
