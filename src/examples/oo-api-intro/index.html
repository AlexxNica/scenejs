Object-Oriented API - Cross-Domain Load of COLLADA Airplane Model from SceneJS.org

SceneJS has had an object-oriented core API since V0.7.3. Layered on top of that is a thin layer of factory functions. Therefore, you have the choice of using the framework in either an object-oriented or functional style. The examples in the API documentation are in the object-oriented style, while many of the examples here use the factory functions for brevity.

Here's the equivalent to the earler Seymour plane example, this time with most of the scene constructed through the composition of objects using the object-oriented API. Note that towards the end of this example we start using the original functional API again, to show how they can be mixed.
