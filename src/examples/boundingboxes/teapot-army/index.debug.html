Description

SceneJS V0.7.0 introduced SceneJS.boundingBox nodes, which specify the spatial boundaries of scene graph subtrees so that they are only traversed when their enclosing boundingBoxes intersect the current view frustum. When configured with a projected size threshold for each child, they can also function as level-of-detail (LOD) selectors, activating a different child as its projected size crosses each threshold.

This example features an array of boundingBox nodes that each selects from among a three child nodes (box, sphere and teapot) for each size threshold as the viewpoint approaches and departs.
