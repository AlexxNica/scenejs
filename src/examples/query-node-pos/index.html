SceneJS > Examples > Scene Query Demo - Get Node Position

This example requires a browser that supports the HTML5 <canvas> feature.

Click a teapot to query its World, View and Canvas-space center point

Click on a teapot, then we'll use the JSON API to walk the nodes on the path to the root while using the matrices we find on each node to determine origin of the teapot in the various coordinate spaces.
