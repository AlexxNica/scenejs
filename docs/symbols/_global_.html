getSafeToCache()
Returns true if the coordinate space defined by the current view matrices is fixed, ie. not animated, at this level of scene traversal. If not, then it is not safe for the client node to memoize its boundary once it is are transformed, because the transformation is therefore likely to vary, causing the boundary to move around.
