1 /**
  2  * @class The basic scene node type, providing the ability to connect nodes into parent-child relationships to form scene graphs.
  3  *
  4  * <h1>Flexible Constructor Signature</h1>
  5  * <p>Node constructors have a flexible signature to support different forms of instantiation. They generally take
  6  * static and/or dynamic configuration arguments followed by zero or more child nodes.</p>
  7  *
  8  * <b>Simple static configuration</b>
  9  * <p>The simplest form is with a single static configuration object. For many nodes you only need to specify properties
 10  * where you want to override the node's defaults. Note the <b>sid</b> property, which is an optional subidentifier
 11  * which must be unique within the scope of the parent {@link SceneJS.Node}:</p>
 12  * <pre><code>
 13  * var n1 = new SceneJS.Scale({
 14  *                 sid:  "myScale",                // Optional subidentifier, unique within scope of parent node
 15  *                 info: "This is my Scale node",  // Optional metadata, useful for debugging
 16  *                 x:    100.0 },                  // Falls back on node's defaults of 1.0 for y and z
 17  *
 18  *                      new SceneJS.Geometry( ... )  // Child nodes, zero or more
 19  *             );
 20  * </code></pre>
 21  * Note the optional <b>info</b> property, which you can provide in order to attach a note that may be useful for
 22  * debugging, which may be got with {@link #getInfo}.
 23  * <h2>Dynamic configuration</h2>
 24  * <p>Dynamic configuration can be achieved through a callback that is invoked each time the node is rendered, which
 25  * will pull configs off the scene data scope (more explanation on that below):</p>
 26  * <pre><code>
 27  * var n2 = new SceneJS.Scale(
 28  *
 29  *                      function(data) {
 30  *                              return {
 31  *                                 x: data.get("scaleX"),    // Falls back on node's default of 1.0 when "scaleX" is null
 32  *                                 y: data.get("scaleY")
 33  *                              };
 34  *                      },
 35  *
 36  *                      new SceneJS.Geometry( ... )
 37  *             );
 38  * </code></pre>
 39  *
 40  * <h2>Static configuration with dynamic override</h2>
 41  * <p>A combination of static and dynamic configuration can be achieved through both a config object and a callback. The
 42  * config object's properties are set on the node immediately, then overridden by the callback at render-time:
 43  * <pre><code>
 44  * var n3 = new SceneJS.Scale(
 45  *
 46  *                      { x: 100.0 },                      // Falls back on node's defaults of 1.0 for y and z
 47  *
 48  *                      function(data) {
 49  *                              return {
 50  *                                 x: data.get("scaleX"),  // Falls back on 100.0 when "scaleX" is null
 51  *                                 y: data.get("scaleY")   // Falls back on node's default of 1.0 when "scaleY" is null
 52  *                              };
 53  *                      },
 54  *
 55  *                      new SceneJS.Geometry( ... ),     // A couple of child nodes this time, just for fun
 56  *                      new SceneJS.Geometry( ... )
 57  *             );
 58  * </pre></code>
 59  *
 60  * <h2>No configuration</h2>
 61  * <p>For many node types you can omit configuration altogether. This node falls back on defaults for all configs:</p>
 62  * <pre><code>
 63  * var n4 = new SceneJS.Scale(                           // Scales by defaults of 1.0 on X, Y and Z axis
 64  *
 65  *                  new SceneJS.Geometry( ... )        // Here's a child node
 66  *             );
 67  * </code></pre>
 68  *
 69  * <h2>A bit more on dynamic configuration</h2>
 70  * <p>The <b>data</b> parameter on the dynamic config callbacks shown above embodies a scene data scope. SceneJS
 71  * provides a fresh global data scope within each scene when it is rendered, into which you can inject data when you
 72  * render the scene graph. The example below demonstrates a property injected into the scope on render, which is then
 73  * pulled by a node's config callback when the node is rendered:
 74  * <pre><code>
 75  * var exampleScene = new SceneJS.Scene({ canvasId: 'theCanvas' },
 76  *
 77  *       new SceneJS.LookAt(
 78  *                      function(data) {
 79  *                            return {
 80  *                                eye  : data.get("eye"),
 81  *                                look : { x: 0, y: 0, z: 0 },
 82  *                                up   : { x: 0, y: 1, z: 0 }
 83  *                            };
 84  *                      },
 85  *
 86  *                      // ... chld nodes ...
 87  *                  );
 88  *
 89  * exampleScene
 90  *     .setData({ eye: { x: 0, y: 0, z: -100 })
 91  *         .render()
 92  * </code></pre>
 93  * <p>Using {@link SceneJS.WithData} nodes, you can create chains of sub-data scopes, to feed data down into the scene
 94  * hierarchy.</p>

135  * <h2>Events</h2>
136  * <p>You can register listeners to handle events fired by each node type. They can be registered either through the
137  * constructor on a static config object, or at any time on a node instance through its {@link #addListener} method.</p>
138  * <p><b>Registering listeners on configuration</b></p>
139  * <p>The example below creates a {@link SceneJS.Instance} node, with a "state-changed" listener registered through its constructor.
140  * See how the listener can have an optional <b>options</b> object, which in this case specifies the JavaScript scope on
141  * which to invoke the handler function.</p>
142  * <pre><code>
143  * var myLoad = new SceneJS.Instance({
144  *
145  *                  uri: "http://foo.com/...",               // File to load
146  *
147  *                  listeners: {
148  *                        "state-changed" : {
149  *                                fn: function(node, params) {
150  *                                       alert("Node " + node.getType() + " has changed state to " + params.newState);
151  *                                    },
152  *                                options: {
153  *                                    scope: this            // Optional scope for handler call, defaults to this
154  *                                }
155  *                         }
156  *                  }
157  *             }
158  *        );
159  * </code></pre>
160  * <p><b>Registering and de-registering listeners on node instances</b></p>
161  * <p>This example registers a "state-changed" listener on an existing instance of the node, then removes it again:</p>
162  * <pre><code>
163  * var handler = function(node, params) {
164  *                  alert("Node " + node.getType() + " has changed state to " + node.getState());
165  *              };
166  *
167  * myLoad.addListener("state-changed", handler,
168  *
169  *              // Listener options
170  *
171  *              {
172  *                    scope: this
173  *              });
174  *
175  * myLoad.removeListener("state-changed", handler);
176  * </code></pre>

178  * @constructor
179  * Create a new SceneJS.Node
180  * @param {Object} [cfg] Static configuration object
181  * @param {function(SceneJS.Data):Object} [fn] Dynamic configuration function
182  * @param {SceneJS.node, ...} arguments Zero or more child nodes
183  */
184 SceneJS.Node = function() {
185     this._nodeType = "node";
186     this._NODEINFO = null;  // Big and bold, to stand out in debugger object graph inspectors
187     this._sid = null;
188     this._children = [];
189     this._fixedParams = true;
190     this._parent = null;
191     this._listeners = {};
192 
193     /* Used by many node types to track the level at which they can
194      * memoise internal state. When rendered, a node increments
195      * this each time it discovers that it can cache more state, so that
196      * it knows not to recompute that state when next rendered.
197      * Since internal state is usually dependent on the states of higher
198      * nodes, this is reset whenever the node is attached to a new
199      * parent.
200      *
201      * private
202      */
203     this._memoLevel = 0;
204     SceneJS.Node._ArgParser.parseArgs(arguments, this);
205 };

209 /**
210  * A simple recursive descent parser to parse SceneJS's flexible node
211  * arguments.
212  *
213  * @private
214  */
215 SceneJS.Node._ArgParser = new (function() {
216 
217     /**
218      * Entry point - parse first argument in variable argument list
219      */
220     this.parseArgs = function(args, node) {
221         node._getParams = function() {
222             return {};
223         };
224         node._fixedParams = true;
225         node._config = {};
226 
227         /* Parse first argument - expected to be either a config object,
228          * config callback or a child node
229          * @private
230          */
231         if (args.length > 0) {
232             var arg = args[0];
233             if (arg instanceof Function) {
234                 this._parseConfigFunc(arg, args, 1, node);
235             } else if (arg._render) {   // Determines arg to be a node
236                 this._parseChild(arg, args, 1, node);
237             } else {
238                 this._parseConfigObject(arg, args, 1, node);
239             }
240         }
241     };
242 
243     /** Parses listeners on a configuration object and registers them on
244      * the given node.
245      * @private
246      */
247     this._parseListeners = function(listeners, node) {
248         for (var eventName in listeners) {
249             if (listeners.hasOwnProperty(eventName)) {
250                 var l = listeners[eventName];
251                 if (!l.fn) {
252                     throw SceneJS._errorModule.fatalError(new SceneJS.errors.InvalidNodeConfigException
253                             ("Listener 'fn' missing in node config"));
254                 }
255                 if (!(l.fn instanceof Function)) {
256                     throw SceneJS._errorModule.fatalError(new SceneJS.errors.InvalidNodeConfigException
257                             ("Listener 'fn' invalid in node config - is not a function"));
258                 }
259                 l.options = l.options || {};
260                 if (!node._listeners[eventName]) {
261                     node._listeners[eventName] = [];
262                 }
263                 node._listeners[eventName].push(l);
264             }
265         }
266     };
267 
268     /**
269      * Parse argument that is a configuration object, then parse the next
270      * argument (if any) at the given index, which is expected to be either a
271      * configuration callback or a child node.
272      * @private
273      */
274     this._parseConfigObject = function(arg, args, i, node) {
275 
276         /* Seperate out basic node configs (such as SID, info and listeners) from other configs - set those
277          * directly on the node and set the rest on an intermediate config object.
278          */
279         for (var key in arg) {
280             if (arg.hasOwnProperty(key)) {
281                 if (key == "listeners") {
282                     this._parseListeners(arg[key], node);
283                 } else if (key == "sid") {
284                     node._sid = arg[key];
285                 } else if (key == "info") {
286                     node._NODEINFO = arg[key];
287                 } else {
288                     node._config[key] = arg[key];
289                 }
290             }
291         }
292 
293         node._getParams = (function() {
294             var _config = node._config;
295             return function() {
296                 return _config;
297             };
298         })();
299 
300         /* Wind on to next argument if any, expected be either
301          * a config callback or a child node
302          */
303         if (i < args.length) {
304             arg = args[i];
305             if (arg instanceof Function) {
306                 this._parseConfigFunc(arg, args, i + 1, node);
307             } else if (arg._render) { // Determines arg to be a node
308                 this._parseChild(arg, args, i + 1, node);
309             } else {
310                 throw SceneJS._errorModule.fatalError(new SceneJS.errors.InvalidNodeConfigException
311                         ("Unexpected type for node argument " + i + " - expected a config function or a child node"));
312             }
313         }
314     };
315 
316     /**
317      * Parse argument that is a configuration callback, then parse
318      * the next argument (if any) at the given index, which is
319      * expected to be a child node.
320      * @private
321      */
322     this._parseConfigFunc = function(arg, args, i, node) {
323         node._getParams = (function() {
324             var _config = node._config;
325             var _arg = arg;
326             var val;
327             return function(data) {
328                 var c = _arg.call(this, data);
329                 if (!c) {
330                     /* Dynamic config returns nothing - we'll assume this is
331                      * explicit, such as when done by a SceneJS.Generator to
332                      * signal the end of its generation.
333                      */
334                     return null;
335                 }
336                 var result = {};
337                 for (var key in _config) {
338                     if (_config.hasOwnProperty(key)) {
339                         result[key] = _config[key];
340                     }
341                 }
342                 for (var key in c) {
343                     if (c.hasOwnProperty(key)) {
344                         val = c[key];   // Don't clobber possible non-null static value with a null dynamic one 
345                         if (val != null && val != undefined) {
346                             result[key] = val;
347                         }
348                     }
349                 }
350                 return result;
351             };
352         })();
353         node._fixedParams = false;
354 
355         /* Wind on to next argument if any, expected be a child node
356          */
357         if (i < args.length) {
358             arg = args[i];
359             if (arg._nodeType) {
360                 this._parseChild(arg, args, i + 1, node);
361             } else {
362                 throw SceneJS._errorModule.fatalError(new SceneJS.errors.InvalidNodeConfigException
363                         ("Unexpected type for node argument " + i + " - expected a child node"));
364             }
365         }
366     };
367 
368     /**
369      * Parse argument that is a child node, then parse the next
370      * argument (if any) at the given index, which is expected to
371      * be a child node.
372      * @private
373      */
374     this._parseChild = function(arg, args, i, node) {
375         node._children.push(arg);
376         arg._parent = node;
377         arg._resetMemoLevel(); // In case child is a pruned and grafted subgraph
378         if (i < args.length) {
379             arg = args[i];
380             if (arg._nodeType) {
381                 this._parseChild(arg, args, i + 1, node);
382             } else {
383                 throw SceneJS._errorModule.fatalError(new SceneJS.errors.InvalidNodeConfigException
384                         ("Unexpected type for node argument " + i + " - expected a child node"));
385             }
386         }
387     };
388 })();

390 /**
391  * Resets memoization level to zero - called when moving nodes around in graph or calling their setters
392  * @private
393  */
394 SceneJS.Node.prototype._resetMemoLevel = function() {
395     this._memoLevel = 0;
396     for (var i = 0; i < this._children.length; i++) {
397         this._children[i]._resetMemoLevel();
398     }
399 };

402 /** @private
403  *
404  * Recursively renders a node's child list. This is effectively rendering a subtree,
405  * minus the root node, in depth-first, right-to-left order. As this function descends,
406  * it tracks in traversalContext the location of each node in relation to the right
407  * fringe of the subtree. As soon as the current node has zero children and no right
408  * sibling, then it must be the last one in the subtree. If the nodes are part of the
409  * subtree of a Symbol node, then a callback will have been planted on the traversalContext
410  * by the Instance node that is intiating it. The callback is then called to render the
411  * Instance's child nodes as if they were children of the last node.
412  */
413 SceneJS.Node.prototype._renderNodes = function(traversalContext, data, children) {
414     var child;
415     var childConfigs;
416     var i;
417     var handle;
418     var savedName;  // Saves SID path for when rendering subgraph of Instance
419 
420     if (this._sidPath) {
421         savedName = SceneJS._instancingModule.getName();      // Save SID path at Instance node
422         SceneJS._instancingModule.setName(this._sidPath);     // Initialise empty SID path for Symbol's subgraph
423     } else if (this._sid) {
424         SceneJS._instancingModule.pushName(this._sid, this);
425     }
426 
427     children = children || this._children;  // for Selector node
428     var numChildren = children.length;
429     if (numChildren) {
430         for (i = 0; i < numChildren; i++) {
431             child = children[i];
432             handle = null;
433             childConfigs = traversalContext.configs;
434             if (childConfigs && child._sid) {
435                 childConfigs = childConfigs[child._sid];
436                 if (childConfigs) {
437                     handle = this._setConfigs(childConfigs, traversalContext.configsModes, child, data);
438                 }
439             }
440             child._renderWithEvents.call(child, { // Traversal context
441                 insideRightFringe: traversalContext.insideRightFringe || (i < numChildren - 1),
442                 callback : traversalContext.callback,
443                 configs: childConfigs || traversalContext.configs,
444                 configsModes : traversalContext.configsModes
445             }, data);
446             if (handle) {
447                 this._unsetConfigs(handle);
448             }
449         }
450     }
451 
452     if (numChildren == 0) {
453         if (! traversalContext.insideRightFringe) {
454 
455             /* No child nodes and on the right fringe - this is the last node in the subtree
456              */
457             if (traversalContext.callback) {
458 
459                 /* The node is within the subtree of a Symbol - Instance has provided a
460                  * callback to render the Instance's child nodes as if they were children
461                  * of the last node in the subtree
462                  */
463                 traversalContext.callback(traversalContext, data);
464             }
465         }
466     }
467     if (savedName) {
468         SceneJS._instancingModule.setName(savedName); // Restore SID path for Instance node
469     } else if (this._sid) {
470         SceneJS._instancingModule.popName();
471     }
472 };

475 SceneJS.Node.prototype._setConfigs = function(childConfigs, configsModes, child, data) {
476     //    var handle = {
477     //        child : child,
478     //        setterFuncs : [],
479     //        values : []
480     //    };
481     var handle = null;
482     var key;
483     var funcName;
484     var func;
485     var config;
486     for (key in childConfigs) {
487         if (childConfigs.hasOwnProperty(key)) {
488             config = childConfigs[key];
489             if (config.isFunc) {
490                 func = child[key];
491                 if (func) {
492                     if (config.value instanceof Function) {
493                         var val = config.value.call(child, data)
494                         func.call(child, val);
495                     } else {
496                         func.call(child, config.value);
497                     }
498                 } else {
499                     if (configsModes && configsModes.strictProperties) {
500                         throw SceneJS._errorModule.fatalError(new SceneJS.errors.WithConfigsPropertyNotFoundException(
501                                 "Method '" + funcName + "' expected on node with SID '" + child.getSID() + "'"));
502                     }
503                 }
504             }
505         }
506     }
507     return handle; // TODO: restore handle!
508 };

510 /**
511  * Wraps _render to fire events either side of rendering.
512  * @private */
513 SceneJS.Node.prototype._renderWithEvents = function(traversalContext, data) {
514     if (this._listeners["rendering"]) { // Optimisation
515         this._fireEvent("rendering", { });
516     }
517     this._render(traversalContext, data);
518     if (this._listeners["rendered"]) { // Optimisation
519         this._fireEvent("rendered", { });
520     }
521 };
522 
523 /** @private */
524 SceneJS.Node.prototype._render = function(traversalContext, data) {
525     if (!this._fixedParams) {
526         this._init(this._getParams(data));
527     }
528     this._renderNodes(traversalContext, data);
529 };

532 // @private
533 SceneJS.Node.prototype._unsetConfigs = function(handle) {
534     for (var i = handle.setterFuncs.length - 1; i >= 0; i--) {
535         handle.setterFuncs[i].call(handle.child, handle.values[i]);
536     }
537 };

539 /** @private */
540 SceneJS.Node.prototype._renderNode = function(index, traversalContext, data) {
541     var child = this._children[index];
542     var childConfigs = traversalContext.configs;
543     if (childConfigs && child._sid) {
544         childConfigs = childConfigs["#" + child._sid];
545         if (childConfigs) {
546             var handle = this._setConfigs(childConfigs, traversalContext.configsModes, child);
547             child._render.call(child, traversalContext, data);
548             this._unsetConfigs(handle);
549             return;
550         }
551     }
552     child._render.call(child, traversalContext, data);
553 };

625 SceneJS.Node.prototype.getNode = function(sid) {
626     for (var i = 0; i < this._children.length; i++) {
627         if (this._children[i].getSID() == sid) {
628             return this._children[i];
629         }
630     }
631     return null;
632 };

634 /** Removes the child node at the given index
635  * @param {int} index Child node index
636  * @returns {SceneJS.Node} The removed child node if located, else null
637  */
638 SceneJS.Node.prototype.removeNodeAt = function(index) {
639     var r = this._children.splice(index, 1);
640     if (r.length > 0) {
641         r[0]._parent = null;
642         return r[0];
643     } else {
644         return null;
645     }
646 };

648 /** Removes the child node with the given SID (structural identifier) string.
649  * @param {String} sid The target child node's SID
650  * @returns {SceneJS.Node} The removed child node if located, else null
651  */
652 SceneJS.Node.prototype.removeNode = function(sid) {
653     for (var i = 0; i < this._children.length; i++) {
654         if (this._children[i].getSID() == sid) {
655             return this.removeNodeAt(i);
656         }
657     }
658     return null;

661 /** Appends a child node
662  * @param {SceneJS.Node} node Child node
663  * @return {SceneJS.Node} The child node
664  */
665 SceneJS.Node.prototype.addNode = function(node) {
666     if (!node) {
667         throw SceneJS._errorModule.fatalError(
668                 new SceneJS.errors.InvalidSceneGraphException(
669                         "SceneJS.Node#addNode - node argument is undefined"));
670     }
671     if (!node._render) {
672         throw SceneJS._errorModule.fatalError(
673                 new SceneJS.errors.InvalidSceneGraphException(
674                         "SceneJS.Node#addNode - node argument is not a SceneJS.Node or subclass!"));
675     }
676     if (node._parent != null) {
677         throw SceneJS._errorModule.fatalError(
678                 new SceneJS.errors.InvalidSceneGraphException(
679                         "SceneJS.Node#addNode - node argument is still attached to another parent!"));
680     }
681     this._children.push(node);
682     node._parent = this;
683     node._resetMemoLevel();
684     return node;
685 };
686 
687 /** Inserts a child node
688  * @param {SceneJS.Node} node Child node
689  * @param {int} i Index for new child node
690  * @return {SceneJS.Node} The child node
691  */
692 SceneJS.Node.prototype.insertNode = function(node, i) {
693     if (node._parent != null) {
694         throw SceneJS._errorModule.fatalError(
695                 new SceneJS.errors.InvalidSceneGraphException(
696                         "Attempted to insert a child to a node without " +
697                         "first removing the child from it's current parent"));
698     }
699     if (i == undefined || i <= 0) {
700         this._children.unshift(node);
701     } else if (i >= this._children.length) {
702         this._children.push(node);
703     } else {
704         this._children.splice(i, 0, node);
705     }
706     node._parent = this;
707     node._resetMemoLevel();
708     return node;
709 };

711 /**
712  * Registers a listener for a given event on this node. If the event type
713  * is not supported by this node type, then the listener will never be called.
714  * <p><b>Example:</b>
715  * <pre><code>
716  * var node = new SceneJS.Node();
717  *
718  * node.addListener(
719  *
720  *              // eventName
721  *              "some-event",
722  *
723  *              // handler
724  *              function(node,      // Node we are listening to
725  *                       params) {  // Whatever params accompany the event type
726  *
727  *                     // ...
728  *              },
729  *
730  *              // options
731  *              {
732  *                     // Optional scope on which handler is
733  *                     // called - default is this
734  *                     scope: this
735  *              }
736  * );
737  *
738  *
739  * </code></pre>
740  *
741  * @param {String} eventName One of the event types supported by this node
742  * @param fn - Handler function that be called as specified
743  * @param options - Optional options for the handler as specified
744  * @return {SceneJS.Node} this
745  */
746 SceneJS.Node.prototype.addListener = function(eventName, fn, options) {
747     var list = this._listeners[eventName];
748     if (!list) {
749         list = [];
750         this._listeners[eventName] = list;
751     }
752     list.push({
753         eventName : eventName,
754         fn: fn,
755         options : options || {}
756     });
757     return this;
758 };
759 
760 /**
761  * @private
762  */
763 SceneJS.Node.prototype._fireEvent = function(eventName, params) {
764     var list = this._listeners[eventName];
765     if (list) {
766         if (!params) {
767             params = {};
768         }
769         for (var i = 0; i < list.length; i++) {
770             var listener = list[i];
771             listener.fn.call(listener.options.scope || this, this, params);
772         }
773     }
774 };

776 /**
777  * Removes a handler that is registered for the given event on this node.
778  * Does nothing if no such handler registered.
779  *
780  * @param {String} eventName Event type that handler is registered for
781  * @param fn - Handler function that is registered for the event
782  * @return {function} The handler, or null if not registered
783  */
784 SceneJS.Node.prototype.removeListener = function(eventName, fn) {
785     var list = this._listeners[eventName];
786     if (!list) {
787         return null;
788     }
789     for (var i = 0; i < list.length; i++) {
790         if (list[i].fn == fn) {
791             list.splice(i, 1);
792             return fn;
793         }
794     }
795     return null;
796 };

799  * @return {SceneJS.Node} this
800  */
801 SceneJS.Node.prototype.removeListeners = function() {
802     this._listeners = {};
803     return this;
804 };

813 /** Returns either all child or all sub-nodes of the given type, depending on whether search is recursive or not.
814  * @param {string} type Node type
815  * @param {boolean} [recursive=false] When true, will return all matching nodes in subgraph, otherwise returns just children (default)
816  * @return {SceneJS.node[]} Array of matching nodes
817  */
818 SceneJS.Node.prototype.findNodesByType = function(type, recursive) {
819     return this._findNodesByType(type, [], recursive);
820 };
821 
822 /** @private */
823 SceneJS.Node.prototype._findNodesByType = function(type, list, recursive) {
824     for (var i = 0; i < this._children; i++) {
825         var node = this._children[i];
826         if (node.nodeType == type) {
827             list.add(node);
828         }
829     }
830     if (recursive) {
831         for (var i = 0; i < this._children; i++) {
832             this._children[i]._findNodesByType(type, list, recursive);
833         }
834     }
835     return list;
836 };
