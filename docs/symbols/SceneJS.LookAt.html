Class SceneJS.LookAt


Extends SceneJS.Node

Usage Example:

Defining perspective, specifying parameters that happen to be the default values


var l = new SceneJS.LookAt({
    eye : { x: 0.0, y: 10.0, z: -15 },
   look : { y:1.0 },
   up : { y: 1.0 },

// .. Child nodes ...
.
A scene node that defines a viewing transform by specifing location of the eye position, the point being looked at, and the direction of "up".
Defined in: lookAt.js.
