Class SceneJS.Translate


Extends SceneJS.Node

Example

A cube translated along the X axis.


var translate = new SceneJS.Translate({
      x: 5.0,
      y: 0.0,
      z: 0.0
  },

     new SceneJS.objects.Cube()
)
.
A scene node that applies a model-space translate transform to the nodes within its subgraph.
Defined in: translate.js.
