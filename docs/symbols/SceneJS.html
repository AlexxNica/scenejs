<static> SceneJS.addListener(name, func)

Adds a listener to be notified when a given event occurs within SceneJS.

Supported events

error

An error has occurred either while defining or rendering a scene. These can be either fatal, or errors that SceneJS can recover from.

Example:


SceneJS.addListener("error", function(e) {
    if (e.exception.message) {
        alert("Error: " + e.exception.message);
    } else {
        alert("Error: " + e.exception);
    }
 });

reset

The SceneJS framework has been reset, where all SceneJS.Scene instances have been destroyed and resources held for them freed.

Example:


 SceneJS.addListener(
     "reset",
     function(e) {
         alert("SceneJS has been reset");
     });

scene-created

A SceneJS.Scene has been defined.

Example:


 SceneJS.addListener(
     "scene-created",
     function(e) {
         alert("A new Scene has been created - scene ID: " + e.sceneId);
     });

scene-rendering

Traversal (render) of a SceneJS.Scene has just begun.

Example:


 SceneJS.addListener(
     "scene-rendering",
     function(e) {
         alert("Rendering of a new Scene has just begun - scene ID: " + e.sceneId);
     });

canvas-activated

A canvas has just been activated for a SceneJS.Scene, where that node is about to start rendering to it. This will come right after a "scene-rendering" event, which will indicate which SceneJS.Scene is the one about to do the rendering.

Example:


 SceneJS.addListener(
     "canvas-activated",
     function(e) {
         var canvas = e.canvas;
         var context = e.context;
         var canvasId = e.canvasId;
         alert("Canvas is about to be rendered to : " + canvasId);
     });

process-created

An asynchronous process has started somewhere among the nodes wtihin a SceneJS.Scene. Processes track the progress of tasks such as the loading of remotely-stored content by SceneJS.Instance nodes. This event is particularly useful to monitor for content loading.

Example:


 SceneJS.addListener(
     "process-created",
     function(e) {
         var sceneId = e.sceneId;
         var processId = e.process.id;
         var timeStarted = e.process.timeStarted;
         var description = e.process.description;
         var timeoutSecs = e.process.timeoutSecs;

         // ...
     });

process-timed-out

An asynchronous process has timed out. This will be followed by a "process-killed" event.

Example:


 SceneJS.addListener(
     "process-timed-out",
     function(e) {
         var sceneId = e.sceneId;
         var processId = e.process.id;
         var timeStarted = e.process.timeStarted;
         var description = e.process.description;
         var timeoutSecs = e.process.timeoutSecs;

         // ...
     });

process-killed

An asynchronous process has finished.

Example:


 SceneJS.addListener(
     "process-killed",
     function(e) {
         var sceneId = e.sceneId;
         var processId = e.process.id;
         var timeStarted = e.process.timeStarted;
         var description = e.process.description;
         var timeoutSecs = e.process.timeoutSecs;

         // ...
     });

scene-rendered

A render of a SceneJS.Scene has completed.

Example:


 SceneJS.addListener(
     "scene-rendered",
     function(e) {
         alert("Traversal completed for Scene - scene ID: " + e.sceneId);
     });

scene-destroyed

A SceneJS.Scene traversal has been destroyed.

Example:


 SceneJS.addListener(
     "scene-destroyed",
     function(e) {
         alert("Scene has been destroyed - scene ID: " + e.sceneId);
     });

<static> SceneJS.setJSONPStrategy(strategy)
Configures SceneJS with a strategy to allow it to use a particular Web service to proxy cross-domain requests by nodes such as SceneJS.Instance when their URIs are not in the local domain. As shown in the example below, the strategy must implement two methods, one to create the request URL and another to extract a string of data from the response.

Example strategy- specifying a JSONP handler for the Yahoo! Query Language (YQL) API:


SceneJS.setJSONPStrategy({
         request : function(url, format, callback) {
                 return "http://query.yahooapis.com/v1/public/yql?q=select%20*%20from%20" + format + "%20where%20url='"
                                  + url + "'&format=" + format + "&callback=" + callback;
          },

         response : function(data) {
                 return data.results[0];
         }
 });

Defined in: loadModule.js.
