{SceneJS.LightSource} setDir(dir)
Sets the light source object-space direction vector. This is only used when the source is of type "dir". Components will fall back on defaults of { x: 0, y: 0, z: -1 } where not supplied; 

myLightSource.setDir({  });       // Sets direction of { x : 0.0, y: 0.0, z: -1.0 }
myLightSource.setDir({ y: 2.0 }); // Sets direction of { x : 0.0, y: 2.0, z: -1.0 }
