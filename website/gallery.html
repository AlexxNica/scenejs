What can SceneJS Do?

Define 3D Scenes Elegantly

SceneJS gives you a big chunk of WebGL functionality up front, with minimal effort. It allows you to create interactive 3D scenes that are compact and expressive, without hiding JavaScript from you. It's also well-suited for learning 3D graphics - you can define a complete scene in one short JavaScript file, while covering a wide range of 3D concepts.

Procedural Scene Generation

Procedural generation involves generating scene content algorithmically rather than manually. SceneJS provides the ability to wrap a subgraph in a function that will loop over it, effectively instancing it multiple times within each frame, each time in a different state.

Example: Generating a Spiral Staircase

Generates a staircase by rotating and translating a flattened cube in a loop, where the generator function varies the rotation angle and translation offset for each iteration.

Interpolated Animation

Almost anything in SceneJS can be interpolated within sequences of key values (usually as a function of time), including transformations, colours, light positions, fog, even other interpolations. While procedural generation is used for spatial generation, interpolation is often used for temporal generation, ie. changing the state of objects over time. The two can be combined for powerful effect, however.

Generalised Level-of-Detail

Level-of-detail (LOD) selection involves decreasing the complexity of a 3D object representation as it moves away from the viewer. This enhances rendering efficiency by decreasing the rendering workload, while the reduced visual quality of the model is often unnoticed because of the small effect on object appearance when distant. SceneJS provides a completely generalised LOD mechanism; for a 3D object, a set of subgraphs, each representing the object at a different level of detail, can be wrapped in an efficient bounding volume so that the approprate subgraph is rendered for the current projected canvas size of the bounding volume.
