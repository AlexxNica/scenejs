About

SceneJS is an open source Javascript framework from XeoLabs that provides a scene graph API on top of the WebGL Canvas element.

SceneJS has some nifty features so far, including:


  • a simple scene definition style
  • an easily extendable API
  • procedural scene generation
  • keyframe animation
  • GLSL shading

Take a look at the gallery of live examples.

And of course - the source code is publically available at GitHub.

SceneJS is maintained by Lindsay Kay, who you can contact at xeolabs@gmail.com
