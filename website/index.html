About

SceneJS is an open source Javascript framework from XeoLabs that provides an extensible scene graph API on top of the powerful OpenGL-specification 3D Canvas element.

SceneJS is currently heading towards it's first release, but already it has a few nifty features, including:


  • a highly extensible API
  • keyframe interpolation animation
  • support for custom OpenGL Shading Language scripts

Read more about those in the ever-growing documentation.

The source code is publically available at GitHub.

SceneJS is maintained by Lindsay Kay, who you can contact at xeolabs@gmail.com
