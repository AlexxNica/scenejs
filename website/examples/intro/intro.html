Introduction

SceneJS is an open source Javascript framework from XeoLabs that provides an extensible scene graph framework on top of the powerful new OpenGL-standard Canvas3D element (and soon the WebGL canvas as well) .

Check out the live example scene graph below, which renders the venerable OpenGL teapot. See how it is composed of functions; these are called in depth-first order, each setting some OpenGL state on entry, then un-setting it again on exit. In this scene, a canvas activates a Canvas element, a shader activates some GLSL scripts, and the rest of the nodes set various matrices, vectors and geometry variables within the scripts (except for viewport which calls a fixed OpenGL function).

Note that the function calls generally take the form node({configs}, child,child...). Node configurations only specify properties that override defaults, such as non-zero vector components.
