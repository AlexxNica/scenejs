SceneJS is an open source Javascript framework from XeoLabs that provides an extensible scene graph API on top of the WebGL Canvas element.

Examples

These are editable live examples showing what SceneJS can do.

Example 1: A Basic Scene Graph

A basic scene graph that renders a teapot with shading. This example shows off projection, viewing and modelling transforms, how to inject data into a scene graph, and the application of GLSL shaders.

Example 2: Procedural Generation

Demonstrates procedural generation of a row of teapots
