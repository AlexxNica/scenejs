SceneJS > Features

Simple

SceneJS makes it easy to become productive with WebGL, allowing you access to a big chunk of its functionality through a terse and expressive pure JavaScript API.

Demo: Hello Teapot!

Efficient

SceneJS uses WebGL efficiently for you, quietly behind the scenes. It caches aggressively to avoid recalculation, automatically generates optimal shaders, intelligently swaps resources in and out of video RAM, and avoids needless repetition of costly WebGL buffer loads by performing them lazily.

Dynamic Content Import

Cross-domain asset import

SceneJS is geared towards composition of scenes from reusable fragments, which a scene can load into itself cross-domain on demand. Its terse API means those fragments carry a lot of functionality for their size, loading and evaluating quickly.

Frustum Culling

SceneJS provides a flexible frustum culling mechanism whereby you may enclose subgraphs in bounding volumes, so that they are only processed when their volumes intersects the view volume.


Level-of-Detail Culling

Level-of-detail (LOD) selection enhances rendering efficiency by decreasing the complexity of a 3D object representation as it moves away from the viewer. SceneJS provides a very flexible LOD mechanism. For a 3D object, a set of subgraphs, each representing the object at a different level of detail, can be wrapped in a bounding volume so that the approprate subgraph is rendered for the current projected canvas size of the bounding volume.

Procedural Scene Generation

Procedural generation involves defining scene content algorithmically rather than manually. SceneJS provides the ability to wrap a subgraph in a function that will loop over it, effectively instancing it multiple times within each frame, each time in a different state.

Demo: Generating a Spiral Staircase

Interpolated Animation

Almost anything in SceneJS can be interpolated within sequences of key values, usually as a function of time. That includes transformations, materials, lighting, even other interpolations.

Demo: Rotating a Teapot using Interpolators
