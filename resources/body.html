SceneJS

SceneJS is a JavaScript framework that allows you to create interactive 3D scenes on WebGL using a terse and flexible API that caters to both object-oriented and functional programming styles.

Contents

Note that SceneJS has an object-oriented core API, with a layer of factory functions on top which provide a handy functional API. Therefore, you have the choice of using the framework in either an object-oriented or functional style. The examples in the API documentation are in the object-oriented style, while many of the examples use the factory functions for brevity. Mentally translating between the two styles is actually quite trivial.
