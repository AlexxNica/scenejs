Beta

SceneJS is a JavaScript framework that allows you to create interactive 3D scenes on WebGL using a terse and flexible API that caters to both object-oriented and functional programming styles.

Pre-Parsed COLLADA Content

One major change in V0.7.6 is the removal of COLLADA parsing from the core framework. That could still be done in the browser using a parser utility, but for speed and efficiency you would probably want to pre-generate a SceneJS JSON content module and load that into your scene. You can do that online with this new tool:

  • Online COLLADA-to-SceneJS Translator

More Data Flow Capabilities

V0.7.6 introduces a new SceneJS.WithConfigs node, which features in many of the budled examples. This node provides a data flow mechanism that is geared towards enabling a server to create/delete/manipulate a client scene graph through the new SceneJS.Socket node.

Contents

Note that SceneJS has an object-oriented core API, with a layer of factory functions on top which provide a handy functional API. Therefore, you have the choice of using the framework in either an object-oriented or functional style. The examples in the API documentation are in the object-oriented style, while many of the examples use the factory functions for brevity. Mentally translating between the two styles is actually quite trivial.
